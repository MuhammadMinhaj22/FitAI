# AI Fitness & Diet Planner Website

## Project Overview
Build a modern, responsive AI Fitness & Diet Planner website.
The website generates a personalized workout plan, diet plan, and weekly schedule based on user input.

This project is frontend-focused and portfolio-ready.

---

## Core Features

### User Input
Collect the following data from the user:
- Age
- Weight (kg)
- Height (cm)
- Gender
- Fitness Goal (lose weight, gain muscle, stay fit)
- Activity Level (low, moderate, high)

---

## Functional Requirements

### Fitness Logic
- Calculate BMI using height and weight
- Suggest daily calorie intake based on goal
- Recommend water intake per day

---

### Workout Plan
Generate a 7-day workout plan:
- Home-based exercises
- Different muscle groups per day
- Include sets, reps, and rest time
- Include 1 rest day

---

### Diet Plan
Generate a 7-day diet plan:
- Breakfast, Lunch, Dinner, Snacks
- Simple and healthy foods
- Estimated daily calories

---

### Weekly Schedule
- Display a day-wise table
- Show workout + diet summary per day

---
